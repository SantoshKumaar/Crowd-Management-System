# Assets Directory

This directory contains all static assets used in the Crowd Management System.

## Structure

- `icons/` - Icon files (SVG, PNG)
- `img/` - Image files (PNG, JPG, SVG)

## Usage

Import assets in your components:

```javascript
import logo from '../assets/icons/logo.svg'
import backgroundImage from '../assets/img/login-bg.png'
```

## Adding Assets

1. Place icon files in `icons/` directory
2. Place image files in `img/` directory
3. Use descriptive file names 
4. Optimize images before adding to reduce bundle size
